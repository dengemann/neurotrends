About NeuroTrends

Thousands of brain imaging studies are published each year. Scientists design and analyze these experiments using a large and growing array of tools. NeuroTrends organizes the brain imaging research literature according to these tools. We search article texts for hundreds of different experimental designs, analysis strategies, and software packages.

To see what NeuroTrends can do, run a search or view the gallery! Or create your own database: all the code behind the NeuroTrends database and this site is available on our GitHub repo.
